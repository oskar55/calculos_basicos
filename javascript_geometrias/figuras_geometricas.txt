CUADRADO
 - Perimetro = sumar todos los lados (L + L + L + L )
 - Area = multiplicar 1 lado por otro (L * L)

TRIANGULO
 - Perimetro = sumar todos los lados (L + L + L )
 - Area = Multiplicar base por altura y dividir esto por 2 (B * A) / 2

CIRCULO
 -  Circunferencia = Diametro * PI (13.1415) 
 -  Diametro = Radio * 2
 -  Area = (Radio * Radio) * PI
